Meta entity
-------

Add content entity based metadata to a content entity of choice.
Primarily meant for content entity that need metatag integrations.
But also can be used for other meta information

Module state: Under heavy development


Requirements
--------------------------------------------------------------------------------
Meta entity for Drupal 8 requires the following:

* Inline Entity Form
  https://www.drupal.org/project/inline_entity_form
  Provides a widget for inline management (creation, modification, removal) of referenced entities.
